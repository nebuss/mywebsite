슬통 자동자는 매해 출시되는 신형 자동차의 에너지 소비효율 등급을 1등급으로 유지하고 있다. 22 년 개발된 신형 모델이 한국 자동차 평가원에서 설정한 에너지 소비 효율등급 1등급을 받을 수 있을지 검정하려한다. 평가원에 따르면 1등급의 기준은 평균 복합 에너지 소비효율이 16.0 이상인 경우 부여 한다고 한다. 다음은 신형 자동차 15대의 복합 에너지소비효율 측정한 결과이다. 15.078, 15.752, 15.549, 15.56, 16.098, 13.277, 15.462, 16.116, 15.214, 16.93, 14.118, 14.927, 15.382, 16.709, 16.804 표본에 의하여 판단해볼때, 현대자동차의 신형 모델은 에너지 효율 1등급으로 판단할 수 있을지 판단해보시오. (유의수준 1%로 설정)

1. 검정을 위한 가설을 명확하게 서술하시오.

#귀무가설 H0: 신형 모델 1등급의 평균 복합 에너지 소비효율은 16.0 이상이다. mu>=16
#대립가설 Ha: 신형 모델 1등급의 평균 복합 에너지 소비효율은 16.0 미만이다. mu < 16

2. 검정통계량 계산하시오.

from sklearn.linear_model import LinearRegression
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy.stats import norm


data = [15.078, 15.752, 15.549, 15.56, 16.098, 13.277, 15.462, 16.116, 15.214, 16.93, 14.118, 14.927, 15.382, 16.709, 16.804]

mu0= 16 # 귀무가설 모평균
sample_std = np.std(data, ddof=1) # 표본표준편차

sample_std
np.float64(0.9802946690129646)
sample_mean = np.mean(data) # 표본평균
sample_mean
np.float64(15.531733333333333)
t_value = (sample_mean - mu0) / (sample_std / np.sqrt(15))

3. p‑value을 구하세요.

from scipy.stats import t
p_value = t.cdf(t_value, df=14)
p_value
np.float64(0.042762417664207845)

4. 현대자동차의 신형 모델의 평균 복합 에너지 소비효율에 대하여 95% 신뢰구간을 구해보세요.

# 신뢰수준에 따른 t 임계값 계산
confidence_level = 0.95
alpha = 1 - confidence_level # 유의수준
t_crit = t.ppf(1 - alpha/2, 14)

# 신뢰구간 계산
margin_of_error = t_crit * (sample_std / np.sqrt(15))
confidence_interval = (sample_mean - margin_of_error, sample_mean + margin_of_error)
confidence_interval
(np.float64(14.988864240339733), np.float64(16.074602426326933))

p-value가 0.04로 유의수준 1%(0.01) 보다 크므로 귀무가설을 기각하지 않는다. 현대 자동차
